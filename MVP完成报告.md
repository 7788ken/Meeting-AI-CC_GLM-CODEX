## 🔧 技术栈

### 前端
- **框架**: Vue 3 + TypeScript
- **状态管理**: Pinia
- **路由**: Vue Router
- **UI 库**: Element Plus
- **音频处理**: Web Audio API
- **实时通信**: WebSocket
- **测试**: Vitest + Playwright

### 后端
- **框架**: NestJS + TypeScript
- **数据库**: PostgreSQL + MongoDB
- **ORM**: Prisma
- **WebSocket**: 原生 WebSocket + Socket.IO
- **AI 服务**: GLM
- **认证**: JWT
- **测试**: Jest

### 部署
- **容器化**: Docker + Docker Compose
- **反向代理**: Nginx
