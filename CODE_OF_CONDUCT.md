# 行为准则

本项目遵循开放、友善、专业的协作原则。我们承诺为所有参与者提供无骚扰的协作环境。

## 我们的承诺

无论年龄、性别、性取向、残障、外貌、种族、宗教、国籍或技术水平如何，所有人都应被尊重。

## 可接受的行为

- 以尊重、耐心、建设性的方式沟通
- 关注事实与技术细节，避免人身攻击
- 接受不同观点并积极寻求共识

## 不可接受的行为

- 侮辱、骚扰、歧视或人身攻击
- 恶意引战、刷屏、煽动对立
- 发布或传播他人隐私信息

## 适用范围

本准则适用于所有项目空间，包括 Issue、PR、讨论区、代码评审、文档与社交渠道。

## 执行与举报

如遇不当行为，请联系项目维护者。建议通过 GitHub 私信仓库管理员，或在需要时使用 GitHub 的私密安全通道进行反馈。维护者将根据情况进行必要处理。

## 归属

本准则参考 Contributor Covenant 的理念进行整理与简化。
